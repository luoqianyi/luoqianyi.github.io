# Typora快捷键

- 无序列表：-  + 空格
- 有序列表：数字 + . +空格
- 标题：ctrl+数字
- 表格：ctrl+t
- 选中一整行:ctrl+l
- 选中单词：ctrl+d
- 选中相同格式的文字：ctrl+e
- 跳转到文章开头:ctrl+home
- 跳转到文章结尾：ctrl+end
- 搜索：ctrl+f
- 替换：ctrl+h
- 引用：> +空格
- 代码块：~~~+回车
- 加粗：ctrl+b
- 倾斜：ctrl+i
- 下划线：ctrl+u
- 删除线：alt+shift+5
- 插入图片：直接拖到指定位置或者ctrl+shift+i
- 插入连接：ctrl+k
